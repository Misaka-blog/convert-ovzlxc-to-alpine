# convert-ovzlxc-to-alpine

重装 OpenVZ 及 LXC 虚拟化架构的 VPS 至 Alpine Linux


```shell
wget -N --no-check-certificate https://raw.githubusercontent.com/Misaka-blog/convert-ovzlxc-to-alpine/main/alpine.sh && bash alpine.sh
```
